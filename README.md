    CE clear data in register

###简易计算器1.0版本<br>
    MC：清除存储的数据<br>
    MR：读取存储的数据<br>
    MS：将所显示的数存入存储器中，存储器中原有的数据被覆盖<br>
    M+：将输入的数加上存储器中的数，所得结果存入存储器<br>
    M-：将存储器中的数减去输入的数，所得结果存入存储器<br>
    CE：在数字输入期间按下此键，将清除输入寄存器中的值并显示"0"，可重新输入<br>
    C：清除全部数据结果和运算符<br>
